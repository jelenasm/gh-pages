Popular Summary

Empirical data about real-world systems hide message about system organization and function that should be revealed. It is often impossible to process all data provided to extract relevant information, but actually we do not have to do that if we apply proper tool. Theory of complex networks provides powerful tool to extract message hidden in the data focusing only on the interactions between elements of the system and neglecting all other details. The idea is to represent elements of the system as nodes and interactions between them as links and then we will have, for example, network of connected individuals, species, proteins, atoms, etc.

Statistical analysis of different real-world networks has shown that elements are not connected randomly, but instead there are structural patterns common to different real systems. If we analyze flow pathways, for example how information moves through a social system, we can see that there are subgroups of nodes where flow stays for relatively long time. This indicates that network has community structure. Detecting community patterns give us important information about organizational principles in the real system.

An important problem, however, is that empirical data we use to construct complex network often contain measurement errors, so that we do not see all interactions. The noise in the network structure can cause incorrect detection of community patterns that further give us misleading information about the system.

To reduce error in community detection in noisy network, we developed method that treats data as incomplete and applies probabilistic approach to estimate flows in the network. The method uses prior assumption of the network with random connections and without community structure. Analysis shows that the proposed method successfully balance the impact of the imposed prior against the observed network data. In the case when there is good signal in the network structure it detects communities correctly. If the level of noise in network is high, it indicates that significant community patterns can not be detected.
